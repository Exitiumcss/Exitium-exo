what this is

exitium is a comp-focused gorilla tag group. we run our own private server so queues aren’t random kids, rules are actually enforced, and scrims feel like scrims — not a public lobby where half the lobby is on tiktok live.

we’re here for people who want real reps: callouts, set plays, rematches, and brackets that don’t fall apart because someone “had to go.” if you like clean 5s and serious 1s, you’ll fit the vibe.

the server

the private server is basically the “best of the best” room — it’s where we host stacked 5v5s, 1v1 ladders, and team events. it’s also where we track results so the win rates below actually mean something (same environment, same rule-set, same people grinding).

  • organized 5s with comms + roles (not chaos)
  • 1v1 blocks for warmups / money matches / seeding
  • staff on deck so ghosting / throwers get clipped fast
